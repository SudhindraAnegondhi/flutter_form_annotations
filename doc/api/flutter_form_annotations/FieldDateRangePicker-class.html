FieldDateRangePicker

FieldDateRangePicker class Null safety

@FieldDateRangePicker() allows picking a range of dates All properties are optional. All dates are 'yyyy-MM-dd' initialDate is set to current date if not provided. firstDate is set to 25 years before current date if not provided. lastDate is set to 25 years after current date if not provided. initialDateRange ia string in "yyyy-MM-dd, yyyy-MM-dd" format specifies the initial date range. if not provided it is set to "firstDate, lastDate""
